Crowdsource
By default RoadSense keeps everything on this device. Upload and download are separate — turning one on does not turn on the other.
Upload my detections
Sends anonymized hazard detections off-device to improve the shared map. Off by default — RoadSense is on-device-only until you enable this.
Download shared hazards
Pulls hazards reported by other drivers so you get warned about hazards you haven't driven over yet. Off by default.
Sync worker URL
Optional. Point upload/download at your own sync worker instead of the default endpoint.
Calibration Data
Delete local calibrations
Removes the hazard detections and calibrations stored on this device. Cannot be undone.
Delete cloud calibrations
Removes the detections you uploaded from the shared cloud map. Other drivers' data is unaffected. Cannot be undone.
